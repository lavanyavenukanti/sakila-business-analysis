# 🎬 Sakila DVD Rental Business Analysis

## 📌 Project Overview

This project analyzes the Sakila DVD Rental database using SQL to answer real-world business questions. The goal is to demonstrate SQL skills used by Data Analysts to generate business insights from relational databases.

---

## 📊 Business Questions Answered

- Which film categories generate the highest revenue?
- Which customers spend the most?
- Which films are rented the most?
- Which stores generate the highest revenue?
- Which actors appear in the most rented films?
- Revenue analysis using SQL queries
- Customer spending analysis
- Rental trends and business insights

---

## 🛠 SQL Skills Used

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- INNER JOIN
- Aggregate Functions
- Subqueries
- Common Table Expressions (CTEs)
