# 🎬 Sakila DVD Rental Business Analysis

## 📖 Project Overview

This project analyzes the **Sakila DVD Rental** database using SQL to answer real-world business questions. The objective is to demonstrate practical SQL skills used by Data Analysts to extract insights, identify business trends, and support data-driven decision-making.

The project focuses on customer behavior, revenue analysis, film inventory, and rental performance while applying industry-standard SQL techniques.

---

## 🎯 Business Objective

The goal of this project is to analyze the DVD rental business and answer important business questions such as:

- Which film categories generate the highest revenue?
- Which customers contribute the most revenue?
- Which films are rented most frequently?
- How does inventory affect rentals?
- What business insights can improve performance?

---

## 🗂️ Dataset

The project uses the **Sakila Sample Database**, a relational database that simulates the operations of a DVD rental store.

The database includes information about:

- Customers
- Rentals
- Payments
- Films
- Categories
- Actors
- Inventory
- Stores
